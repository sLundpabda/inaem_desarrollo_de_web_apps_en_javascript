# CURSO DE DESARROLLO WEB EN JAVASCRIPT DEL INAEM
Un curso de aplicaciones web con JS y TypeScript, más la iniciación de Angular 19, 
que se cursa desde el Lunes 10 de Noviembre de 2025, hasta el Martes 25 de Noviembre de 2025 (sin contar los Viernes de las 2 semanas).
El horario es a las 17:00 hasta las 21:00 en Zaragoza (CTA: Centro de Tecnologias Avanzadas, INAEM).
